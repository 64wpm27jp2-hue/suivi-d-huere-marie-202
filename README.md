# Mes Heures Apprenti

Application web permettant de calculer le salaire d'un apprenti.

## Fonctionnalités

- Calcul du taux horaire
- Calcul du salaire mensuel
- Heures de nuit
- Heures du dimanche
- Jours fériés
- Heures supplémentaires

## Utilisation

Ouvrir le fichier index.html dans un navigateur.
